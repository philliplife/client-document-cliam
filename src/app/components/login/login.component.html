Instruction to set up environment.
  • Open UPS2 system.
  • Register new project.
  • Create new tasks for the project as follows:
    "HOME", "MAIN"
  • Create new groups for the project
    and assign tasks HOME, MAIN to this groups.
  • Register new system (appCode is same as 'project id').
  • Open file environment.ts and environment.prod.ts
  • Place appCode ex. appCode: 'UPS2'
  • Place system_id in environment.ts
    ex. system_id: '7CC01F99-4BA8-4B71-AD1F-7B15A4A63761'
